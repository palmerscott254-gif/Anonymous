Client folder scaffold. Move your existing UI files into `client/src/`.

Suggested structure:
- `src/components/` - UI components
- `src/pages/` - Route pages
- `src/hooks/` - Custom hooks
- `src/services/` - API and socket wrappers
- `src/store/` - Zustand/Redux stores
- `src/utils/` - Utilities

After moving, update imports to match the new paths.
